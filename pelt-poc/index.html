Same Pelt, Different Content

Multiple elements can share the same pelt definition. The SVG stretches to fill the element's bounding rect.

A nature-inspired frame with dashed borders and corner accents. This visual cannot be achieved with CSS border properties alone.

When the engine integration is complete, CSS custom properties (--pelt-*) will flow into pelt SVGs via var() references, enabling per-section theming.
